Getting Started with Windsurf

Option 1: Windsurf IDE

  1. Visit codeium.com/windsurf
  2. Download for Windows/Mac/Linux
  3. Install like any application
  4. Sign up for account
  5. Start coding with AI!

Best for: Clean start, AI-first development

Option 2: VS Code Plugin

  1. Open VS Code
  2. Go to Extensions (Ctrl+Shift+X)
  3. Search "Windsurf"
  4. Click Install
  5. Sign in to activate

Best for: Existing VS Code users

Free Trial: Both options include a free trial to test all features!
